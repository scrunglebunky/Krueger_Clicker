--STUDENT IMAGES--

For added customization, you can drag and drop images into the "students" folder to have your photos in the game.

No names are necessary as they are randomly selected.

This can lead to errors if invalid files are dropped into the folders. 
Please be nice, this is a game meant for everyone and ruining it for everyone is almost pointless.


--SAVE FILES--

You are able to save your game into a .KROOG file to be loaded in the future. 

When saving a game, a box will open in the terminal. 
Simply type in a save file name and it will be saved into the "saves" folder.

When loading a game, an input will open in the terminal.
Simple type in a save file name and it will be opened from the "saves" folder. CAP SENSITIVE.
If an invalid name is entered, the terminal will tell you and the game will skip over the load.


--ERROR HANDLING--

If the game fails on startup, run the "repair.py" file to fix the errors.
"repair.py" loads every image, every sound, every file specifically, and deletes them if they are erroneous.
"repair.py" will also detect missing files and will alert you of what is missing.
